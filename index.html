Steam Games Analysis Full Project

The project aimed to extract and analyze Steam data, starting from a small sample of 3 Steam IDs and 152 games, expanding to 7,000 Steam IDs and over 13,000 games across 23 genres. Using a pipeline that combined API calls, web scraping, and data cleaning, the data was uploaded to BigQuery for SQL analysis and subsequently visualized in Tableau. The objectives were to identify the most popular games and genres by user base and playtime. Two dashboards showcasing these insights were successfully created, meeting all project goals. .
